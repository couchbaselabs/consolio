Sync up Your Mobile Data

Couchbase Cloud is a free service for Couchbase Lite users and other developers to get started with Couchbase Sync Gateway and Couchbase Server.

JSON Anywhere

JSON Anywhere means: JSON on the device, JSON on the wire, and JSON in the cloud. Data stored in Couchbase Lite on a mobile device is seamlessly synchronized with the Couchbase Server in the cloud, enabling the simple development of a new class of occasionally connected, data intensive mobile applications.

Couchbase Server is a high performance NoSQL Document Database. The flexible data model, easy scalability and sub-millisecond performance make it an easy choice for demanding webscale applications as well as rapid development required by startups.
Couchbase Sync Gateway enables easy, reliable data sync to the cloud. Built on a shared nothing architecture, the Couchbase Sync Gateway provides the extremely high scalability required by mobile applications and solves complex distributed data synchronization, including conflict resolution.
Couchbase Lite is a secure, ultra-lightweight, native JSON-based NoSQL database built for the online/offline mobile world. It is the first native NoSQL database for mobile devices, providing native cross-platform support for iOS, Android and REST/HTML5.
