Couchbase Cloud FAQ

I. Overview

Q: What is Couchbase Cloud?
Couchbase Cloud is a free Couchbase Server and Sync Gateway instance that helps Couchbase Lite developers up and running quickly.
Q: How does it work?
It’s free! Couchbase Lite, our embedded JSON database for mobile was designed with sync capability in mind. Couchbase Lite works with Couchbase Sync Gateway, a sync tier that serves as an extension of our award-winning NoSQL database, Couchbase Server.

On Couchbase Cloud, developers can start a Sync Gateway and Couchbase Server instance. Couchbase Lite-backed mobile apps that need to share, collaborate or securely back up data to a back-end store can use the Sync Gateway REST APIs to pull and push data to Couchbase Server.

Play around with Couchbase Cloud to see what a full Couchbase service stack can do for you.

II. Getting Started

Q: How do I sign-up?
It’s easy. Couchbase Cloud uses Mozilla Persona to create and authenticate accounts. Sign-in with just a valid e-mail address, and a password. That’s it! Start your registration here to get your own Couchbase Cloud today.
Q: How do I create and connect to Couchbase Cloud Sync Gateways?
After you’ve opened your Couchbase Cloud account, head over to your Dashboard. Click the “New Sync Gateway” button to create a new Couchbase Cloud Sync Gateway. This opens a Sync Gateway configuration window.

Give your Sync Gateway a name, and add your custom Sync Function code. If you are unfamiliar with the Sync Function APIs, check out the documentation here. Below is an example and breakdown of some Sync Function code. 
{
    "interface":":4984",
    "adminInterface":":4985",
    "log":["REST"],
    "databases":{
    "sync_gateway":{
        "server":"http://localhost:8091",
        "bucket":"sync_gateway",
        "sync":`function(doc) {channel(doc.channels);}`
    }
}
The local database is called sync_gateway and is served from port 4984, with the admin interface on port 4985.

Click the “Create” button to submit the configuration. The Sync Gateway URL is displayed. Use that URL for Sync REST API requests.

III. Couchbase Cloud Account Management

Q: How do I change my Couchbase Cloud username or password?
Our account authentication is handled by Mozilla Persona, with each account username tied to a valid e-mail address. While username updating is limited in Persona, you can easily update your password at login.persona.org.
Q: Can I connect directly to my Couchbase Cloud Couchbase Server instances instead of through Sync Gateway?
It’s an offering that we’re actively thinking about, but in the meantime if you’re looking for access to Couchbase Servers, we recommend checking out our Amazon Marketplace offerings, or download Couchbase Server for your in-house use.
Q: Are there REST APIs for Sync Gateway?
The sync is performed using REST calls from Couchbase Lite to Sync Gateway. For more information about the sync REST APIs, check out our Sync Gateway documentation here. We also have REST APIs for Sync Gateway administration.
Q: Can I run commands to Couchbase Cloud through REST APIs?
Yes! You can run commands to your Couchbase Cloud Sync Gateway, or any Sync Gateway, using our Admin REST APIs. First, find your Couchbase Cloud Sync Gateway’s Admin REST API port by going to your Dashboard, and clicking on your Sync Gateway instance’s “Admin” URL link. Using this URL, and the curl tool you can send REST API commands to your Sync Gateway.

IV. Security

Q: How secure is my data?
Couchbase Cloud is a great prototyping tool to help developers get a sense of what its like to work with Couchbase Server, and Couchbase Sync Gateway. We recommend hosting your secure data on an in-house Couchbase Server instance, or on AWS via our Amazon Marketplace offerings.

V. Hosting, Backups & Exporting

Q: Where does Couchbase Cloud host my data?
Couchbase Cloud is hosted on Amazon (AWS).

When you are ready to go into production, we offer an exclusive Couchbase Server and Sync Gateway bundle for free on the Amazon Marketplace. We also offer a variety of Couchbase Server Enterprise Edition a la carte options on Amazon Marketplace, as well as downloads of both Couchbase Server and Sync Gateway on our website.
Q: How can I backup my data?
Couchbase Cloud is a great prototyping tool to help you get a sense of what its like to work with Couchbase Server, and Couchbase Sync Gateway. We recommend using a production-quality Couchbase Server for your regular and backup needs. For cloud-based solutions, check our Amazon Marketplace offerings, or if you’re looking to move your data in-house, download Couchbase Server and Sync Gateway from our website.
Q: How do I export my data?
Get in touch with us to export your Couchbase Cloud data.

For Couchbase Lite developers who want to migrate to their own Couchbase Server and Sync Gateway, we also recommend simply syncing from your Couchbase Cloud account to your Couchbase Lite client, and then syncing from your Couchbase Lite client to your Sync Gateway tied to a Couchbase Server instance.
Q: How do I restart my server?
If you are having server issues, get in touch with us and we will investigate the problem. If a restart is needed, we will let you know and coordinate an appropriate downtime.

VI. Scaling & Production

Q: Is there a limit to the number of Sync Gateways I can create?
Each of our Couchbase Cloud accounts host up to 500MB to give developers the freedom to prototype what they need. When you’re ready to go to production, check out our Amazon Marketplace offerings or download Couchbase Server and/or Sync Gateway for your in-house needs.
Q: Can I use Couchbase Cloud for production?
No. Couchbase Cloud is a sandbox service, and is our premier prototyping tool for developers interested in a hands-on learning experience for Sync Gateway and Couchbase Server. When you’re ready to go to production, check out our Amazon Marketplace offerings or download Couchbase Server and/or Sync Gateway for your in-house needs.

None at the moment.

VIII. Support

Q: Does Couchbase Cloud have support?
We’re currently in alpha, and do not have formal support offerings, but we would love to hear feedback and questions from the developer community. Feel free to get in touch with us right here.

We also recommend checking out our Couchbase Server Enterprise Edition, which comes with a variety of full service options. Learn more here.
Q: What version of Couchbase Server does Couchbase Cloud use?
Couchbase Cloud uses a specialized hosting edition of Couchbase Server that is compatible with Couchbase Server 2.0.
Q: What version of Sync Gateway does Couchbase Cloud use?
We are currently using Sync Gateway 1.0-Beta.
Q: How do I delete my account?
You can safely delete your databases at any time. If you need to delete your account, just let us know.
